# 水果消消乐项目计划书

## 一、项目简介

### 1.1 项目名称

**水果消消乐（Fruit Crush）**

### 1.2 项目概述

水果消消乐是一款基于经典宝石迷阵玩法的休闲益智消除游戏。玩家通过交换相邻的水果，使 3 个或更多相同类型的水果连成一线进行消除，获得分数。游戏融合了特殊元素、道具系统、成就系统和多种游戏模式，提供丰富的游戏体验。

### 1.3 技术栈

| 类别     | 技术选型                  |
| -------- | ------------------------- |
| 开发语言 | C++                       |
| 图形框架 | Qt 6.x + OpenGL           |
| 构建工具 | CMake / qmake             |
| 版本控制 | Git                       |
| 数据存储 | SQLite（本地排行榜/成就） |

### 1.4 开发周期

预计开发周期：4 周

| 阶段    | 时间           | 内容                               |
| ------- | -------------- | ---------------------------------- |
| 第 1 周 | 需求分析与设计 | 完成需求文档、系统设计、UI 原型    |
| 第 2 周 | 核心功能开发   | 游戏地图、基础消除、得分系统       |
| 第 3 周 | 拓展功能开发   | 特殊元素、道具、动画效果           |
| 第 4 周 | 完善与测试     | 赛制、成就系统、Bug 修复、答辩准备 |

### 3.4 赛制模式设计

| 模式         | 规则                         | 特点                                                 |
| ------------ | ---------------------------- | ---------------------------------------------------- |
| **休闲模式** | 无时间限制，通过消除获得点数 | 主打成就收集；点数可兑换道具；解锁全部成就为终极目标 |
| **比赛模式** | 限时 60 秒/90 秒/120 秒      | 固定道具配给；计入排行榜；追求高分                   |

#### 休闲模式成就玩法核心设计

休闲模式以**成就收集**为核心驱动力，玩家通过完成各类成就获取点数奖励，点数可用于：

- 兑换道具（锤子、夹子、魔法棒）
- 解锁皮肤/主题（可选拓展）
- 展示成就徽章

**成就进度系统**：

- 成就分为 **青铜 → 白银 → 黄金 → 钻石** 四个等级
- 高等级成就需要在低等级基础上继续挑战
- 全成就解锁后获得「水果大师」称号

### 4.3 数据结构设计

#### 4.3.1 核心数据结构

```cpp
// 水果类型枚举
enum class FruitType {
    APPLE,      // 苹果
    ORANGE,     // 橙子
    GRAPE,      // 葡萄
    BANANA,     // 香蕉
    WATERMELON, // 西瓜
    STRAWBERRY, // 草莓
    EMPTY       // 空位
};

// 特殊类型枚举
enum class SpecialType {
    NONE,           // 普通水果
    LINE_H,         // 横向直线炸弹
    LINE_V,         // 纵向直线炸弹
    DIAMOND,        // 菱形炸弹
    RAINBOW         // 万能炸弹（彩虹果）
};

// 水果结构体
struct Fruit {
    FruitType type;           // 水果类型
    SpecialType special;      // 特殊类型
    int row, col;             // 位置坐标
    bool isMatched;           // 是否被匹配
    bool isMoving;            // 是否正在移动
    float animationProgress;  // 动画进度 [0.0, 1.0]
};

// 游戏地图 - 8x8二维数组
const int MAP_SIZE = 8;
Fruit gameMap[MAP_SIZE][MAP_SIZE];

// 或使用动态数组
```

#### 4.4.1 图案匹配算法（BFS/扫描法）

```cpp
/**
 * @brief 检测地图中所有匹配
 * @return 所有匹配结果列表
 *
 * 算法步骤：
 * 1. 横向扫描：遍历每行，统计连续相同水果
 * 2. 纵向扫描：遍历每列，统计连续相同水果
 * 3. 合并重叠：将交叉点的匹配合并（用于检测L/T形）
 * 4. 生成结果：确定每个匹配应生成的特殊元素
 */
std::vector<MatchResult> detectMatches() {
    std::vector<MatchResult> results;
    bool matched[MAP_SIZE][MAP_SIZE] = {false};

    // 横向扫描
    for (int row = 0; row < MAP_SIZE; row++) {
        int count = 1;
        for (int col = 1; col <= MAP_SIZE; col++) {
            if (col < MAP_SIZE &&
                gameMap[row][col].type == gameMap[row][col-1].type &&
                gameMap[row][col].type != FruitType::EMPTY) {
                count++;
            } else {
                if (count >= 3) {
                    // 记录匹配位置
                    MatchResult match;
                    match.fruitType = gameMap[row][col-1].type;
                    for (int i = 0; i < count; i++) {
                        match.positions.push_back({row, col-1-i});
                        matched[row][col-1-i] = true;
                    }
                    // 确定特殊元素类型
                    match.generateSpecial = determineSpecialType(count, true);
                    results.push_back(match);
                }
                count = 1;
            }
        }
    }

    // 纵向扫描（类似逻辑）
    // ...

    // 合并L/T形匹配
    mergeIntersections(results);

    return results;
}
```

#### 4.4.2 消除与下落算法

```cpp
/**
 * @brief 执行消除并处理下落
 * @param matches 待消除的匹配列表
 *
 * 算法步骤：
 * 1. 标记消除：将匹配位置标记为EMPTY
 * 2. 计算下落：从下往上扫描，计算每个水果需要下落的格数
 * 3. 生成新水果：在顶部空位生成新水果
 * 4. 触发动画：将下落和生成加入动画队列
 */
void eliminateAndFall(const std::vector<MatchResult>& matches) {
    // 1. 消除匹配的水果
    for (const auto& match : matches) {
        for (const auto& pos : match.positions) {
            gameMap[pos.first][pos.second].type = FruitType::EMPTY;
        }
        // 在匹配中心生成特殊元素
        if (match.generateSpecial != SpecialType::NONE) {
            auto center = match.positions[match.positions.size()/2];
            generateSpecialFruit(center.first, center.second,
                                 match.fruitType, match.generateSpecial);
        }
    }

    // 2. 处理下落
    for (int col = 0; col < MAP_SIZE; col++) {
        int emptyCount = 0;
        for (int row = MAP_SIZE - 1; row >= 0; row--) {
            if (gameMap[row][col].type == FruitType::EMPTY) {
                emptyCount++;
            } else if (emptyCount > 0) {
                // 下落
                gameMap[row + emptyCount][col] = gameMap[row][col];
                gameMap[row][col].type = FruitType::EMPTY;
            }
        }
        // 3. 顶部生成新水果
        for (int i = 0; i < emptyCount; i++) {
            gameMap[i][col] = generateRandomFruit(i, col);
        }
    }

    // 4. 递归检测新匹配（连锁消除）
    auto newMatches = detectMatches();
    if (!newMatches.empty()) {
        matchQueue.push(newMatches);
    }
}
```

#### 4.4.3 特殊元素效果算法

```cpp
/**
 * @brief 触发特殊元素效果
 * @param row 行坐标
 * @param col 列坐标
 * @return 受影响的位置列表
 */
std::vector<std::pair<int,int>> triggerSpecialEffect(int row, int col) {
    std::vector<std::pair<int,int>> affected;
    SpecialType special = gameMap[row][col].special;

    switch (special) {
        case SpecialType::LINE_H:
            // 消除整行
            for (int c = 0; c < MAP_SIZE; c++) {
                affected.push_back({row, c});
            }
            break;

        case SpecialType::LINE_V:
            // 消除整列
            for (int r = 0; r < MAP_SIZE; r++) {
                affected.push_back({r, col});
            }
            break;

        case SpecialType::DIAMOND:
            // 消除5x5菱形范围
            for (int dr = -2; dr <= 2; dr++) {
                for (int dc = -2; dc <= 2; dc++) {
                    if (abs(dr) + abs(dc) <= 2) {  // 菱形条件
                        int nr = row + dr, nc = col + dc;
                        if (isValidPosition(nr, nc)) {
                            affected.push_back({nr, nc});
                        }
                    }
                }
            }
            break;

        case SpecialType::RAINBOW:
            // 消除所有同类型水果
            FruitType targetType = findSwapTarget(row, col);
            for (int r = 0; r < MAP_SIZE; r++) {
                for (int c = 0; c < MAP_SIZE; c++) {
                    if (gameMap[r][c].type == targetType) {
                        affected.push_back({r, c});
                    }
                }
            }
            break;
    }

    // 检查被影响位置中是否有其他特殊元素（连锁引爆）
    for (const auto& pos : affected) {
        if (gameMap[pos.first][pos.second].special != SpecialType::NONE) {
            auto chainEffect = triggerSpecialEffect(pos.first, pos.second);
            affected.insert(affected.end(), chainEffect.begin(), chainEffect.end());
        }
    }

    return affected;
}
```

#### 4.4.4 得分计算算法

```cpp
/**
 * @brief 计算消除得分
 * @param matches 匹配结果
 * @param comboCount 连击数
 * @return 本次消除总得分
 */
int calculateScore(const std::vector<MatchResult>& matches, int comboCount) {
    int totalScore = 0;

    for (const auto& match : matches) {
        int baseScore = match.positions.size() * 10;  // 基础分：每个10分

        // 额外消除加成
        if (match.positions.size() == 4) {
            baseScore += 20;  // 4消额外20分
        } else if (match.positions.size() == 5) {
            baseScore += 50;  // 5消额外50分
        } else if (match.positions.size() > 5) {
            baseScore += 50 + (match.positions.size() - 5) * 30;
        }

        // 特殊元素加成
        if (match.generateSpecial != SpecialType::NONE) {
            baseScore *= 1.5;
        }

        totalScore += baseScore;
    }

    // 连击加成：combo * 20%
    totalScore = totalScore * (1.0 + comboCount * 0.2);

    return totalScore;
}
```

#### 5.3.5 连锁消除动画序列

```
时间轴 (ms)
0      200    400    600    800    1000   1200
├──────┼──────┼──────┼──────┼──────┼──────┤
│ 消除1 │ 下落1 │ 消除2 │ 下落2 │ 消除3 │ 稳定 │
│      │      │ 连击+1│      │ 连击+2│      │
└──────┴──────┴──────┴──────┴──────┴──────┘

每次连锁消除：
1. 消除动画 (250ms) - 匹配水果同时消失
2. 得分飘字 (600ms) - 与消除同时开始
3. 下落动画 (150ms × 下落格数)
4. 新水果生成 (180ms)
5. 检测新匹配 → 如有则重复步骤1
```
